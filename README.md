# Ford-Car-Prices
Built regression models to predict used Ford car prices using Python, performing data cleaning, feature encoding, and exploratory analysis. Implemented single, multiple linear and ridge regression from scratch and evaluated performance using R² and MSE.

Dataset

Source: IBM Skills Network

Format: CSV file loaded using pandas

Key features:

year

mileage

tax

mpg

engineSize

fuelType

transmission

price (target variable)

🛠️ Tools & Technologies

Python

pandas – data manipulation

NumPy – numerical computations

Matplotlib – visualization

Seaborn – statistical plots
